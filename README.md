# TodoAPP - TodoApp for study of React

App for study react

# BackEnd
  - NodeJS
  - MongoDB
  - Node-restful
  - Mongoose
  - PM2
  - Nodemon

# FrontEnd
  - HTML
  - Sass
  - Javascript
  - Webpack
  - React
  - Redux
  - Bootstrap

### Installing

Install mongodb and nodejs

After install node and mongo, open connection with mongo

If the variable environment of mongodb is ok

Open terminal and write

```
./mongod
```


### Start back end

After git clone

Go to folder "backend"

```
cd backend
```

And install dependencies

```
npm i
```

And run

```
npm run dev
```


### Start front end

Go to folder "frontend"

```
cd frontend
```

And install dependencies

```
npm i
```

Run in mode of development

```
npm run dev
```

Run in mode of production

```
npm run prod
```


And open browser: http://localhost:8081
